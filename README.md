# atelier-telegram-bot.


This is a simple Telegram bot that answers common questions about Atelier.

## Features
- Answers questions like:
  - What is Atelier?
  - How to register
  - How payments work
  - How to list services

## Deployment
1. Create a Telegram bot via [BotFather](https://t.me/botfather) and get the token.
2. Replace `BOT_TOKEN` in `bot.py` with your token.
3. Push this repo to GitHub.
4. Deploy on **Railway** or **Render**:
   - Railway: https://railway.app/
   - Render: https://render.com/

Your bot will run 24/7 without needing a PC.

## Usage
- Start your bot: `/start`
- Ask a question like: "How do I register?"
